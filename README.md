# PDFNOVA

  A premium browser-based PDF tools platform with 28+ tools, all processing client-side.

  ## Features

  - 28+ PDF tools (merge, split, compress, convert, edit, and more)
  - 100% client-side processing — your files never leave your browser
  - Powerful animations inspired by Apple's design language
  - Dark/light mode toggle
  - Vibrant violet + cyan dual-color scheme
  - Completely free — no pricing, no Pro plan, no usage limits

  ## Tech Stack

  - React + TypeScript + Vite
  - Framer Motion for animations
  - TailwindCSS + shadcn/ui
  - PDF.js for PDF processing
  - Express.js backend

  ## Getting Started

  ```bash
  npm install
  npm run dev
  ```

  Open [http://localhost:5000](http://localhost:5000) to view in your browser.
